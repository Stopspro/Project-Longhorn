The World's Smallest OS.

Longhorn is compact.

Project Longhorn is an operating system that is designed to be tiny. At a current size of (enter ISO size) megabytes, we are the smallest OS around.

Longhorn doesn't compact the code into huge, dense portions either. By using ALCAP (as little code as possible), we get stuff done and keep the file size low.

Longhorn is open-source.
